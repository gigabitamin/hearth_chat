### ✅ Python으로 팩토리얼 구현하기

다음은 재귀 함수로 팩토리얼을 구현하는 예제입니다:

```python
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 120
```

### 📊 시간 복잡도 비교

| 알고리즘 | 시간복잡도 | 공간복잡도 |
|----------|-------------|-------------|
| 재귀 팩토리얼 | O(n) | O(n) |
| 반복문 팩토리얼 | O(n) | O(1) |
